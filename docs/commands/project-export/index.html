rundeck-admin

project-export

Export a project.

SYNOPSIS

rerun rundeck-admin:project-export --project <> --url <> --user <> --password <> [--archive <>]

OPTIONS

  • --project <>: the project name.
  • --url <>: rundeck server url.
  • --user <>: the login user name.
  • --password <>: the login password.
  • [--archive <>]: the archive file.

README

Use project-export to export a project into an archive you can import using project-import. If no --archive option is specified one will be written to the current working directory.

Examples

Export a project called "qa1" to a file "qa1.archive":

rerun rundeck-admin:project-export --project qa1 --admin qa1.archive \
    --url http://rundeck1:4440 \
    --user admin --password admin
